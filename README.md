# React App Cart

## Installation and run

```
1. Copy repo
2. cd store-react
3. yarn
4. yarn run json-server to start a local server
5. yarn start
```

## Tech stack

```
1. React, React Hooks, React Router
2. Redux, Redux Saga, Redux Persist
3. Formik, yup
4. React Bootstrap, styled-components
5. json-server
```
